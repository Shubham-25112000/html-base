CSS Styling - Making It Beautiful

CSS Properties and Selectors:

/* Styling the body */
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 0;
    padding: 20px;
}

/* Styling headings */
h1 {
    color: #333;
    text-align: center;
    border-bottom: 2px solid #6366f1;
}

/* Button styling */
.button {
    background-color: #6366f1;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

CSS Layout Techniques:

  • Flexbox: One-dimensional layouts (rows/columns)
  • Grid: Two-dimensional layouts (rows and columns)
  • Responsive Design: Adapts to different screen sizes
  • Media Queries: Different styles for different devices

JavaScript Interactivity

Adding Dynamic Behavior:

// Variables and Functions
let userName = 'Welcome';
const button = document.querySelector('.button');

function greetUser() {
    const name = prompt('What is your name?');
    document.querySelector('h1').textContent = `Hello, ${name}!`;
}

// Event Listeners
button.addEventListener('click', greetUser);

// DOM Manipulation
document.querySelector('.container').style.backgroundColor = 'lightblue';
